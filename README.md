# Vesta CFO

React + Vite front end for Vesta CFO (hotel financial intelligence). Supabase powers auth, data, and edge functions.

## Requirements

- Node.js 20+
- npm (lockfile: `package-lock.json`)

## Commands

```bash
npm ci
npm run dev      # local dev server
npm run test     # Vitest
npm run build    # production build
npm run lint     # ESLint
```

Email and auth templates for Supabase are documented in `SUPABASE_EMAIL_SETUP.md`.
